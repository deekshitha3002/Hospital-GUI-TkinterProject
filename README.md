# Prathima Hospital Appointment System 

A Python Tkinter based hospital appointment booking system.

## Features
- User Registration & Login
- Department & Doctor selection
- Appointment time adjustment
- Image-based UI
- Appointment history table

## Technologies Used
- Python
- Tkinter

## How to Run
1. Install Python (3.x)
2. Run:
   python main.py
